# Module icons

Place module logo images here so the ServicesCarousel and other components can load them. If files are missing, the UI shows a fallback (first letter of the module name with the module color).

Expected filenames (from `app/lib/modules.ts`):

- `main-organic.png`, `main-3d.png`
- `care-organic.png`, `care-3d.png`
- `transport-organic.png`, `transport-3d.png`
- `jobs-organic.png`, `jobs-3d.png`
- `foods-organic.png`, `foods-3d.png`
- `moves-organic.png`, `moves-3d.png`
- `marketplace-organic.png`, `marketplace-3d.png`
- `kids-organic.png`, `kids-3d.png`

Recommended size: 56×56 px or larger (displayed at 56×56 in the carousel).
